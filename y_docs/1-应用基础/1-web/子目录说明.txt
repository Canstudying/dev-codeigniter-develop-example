================================
web子目录说明：
================================
app		应用目录
	cache		框架缓存
	config		配置文件
	controllers	控制器
	core		这个暂没用上
	helpers		这个暂没用上
	hooks		这个暂没用上
	language	这个暂没用上
	libraries	这个暂没用上
	logs		框架日志
	models		模型
	third_party	这个暂没用上
	views		视频
public 	访问入口，只开放这个目录给nginx
	css			样式文件
	js			JS文件
	file		文件,如pdf、excel、word等
	image		图片,用户上传的图片
storage
	data		业务数据、缓存数据
	logs		业务日志、系统日志、API日志等
